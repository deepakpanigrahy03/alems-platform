CPU & C-State Analysis IPC · frequency · residency · thermal —
C-State Residency — measured from cpu_samples
higher C6/C7 = deeper sleep = more efficient idle
Cloud runs: CPU mostly in C6/C7 (deep sleep) between API calls — low power, high idle efficiency. Local runs: forced C0 (active) throughout — inference loop keeps all cores busy, high power draw.

OS Scheduler Analysis Thread migrations · context switches · IRQ rates
Thread migrations vs Duration
Strong linear relationship (r²≈0.89). Duration is the primary driver — longer agentic runs generate migration bursts at phase transitions.
Voluntary vs Involuntary context switches
Involuntary preemptions spike in local inference — kernel preempts the model thread, causing cache state loss and contributing to elevated miss rates.
